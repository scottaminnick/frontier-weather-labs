Search for an airport by name, city, or enter ICAO code directly (e.g., KMCI, KJFK, KLAX)
⚠️ DISCLAIMER: LAMP (Localized Aviation MOS Program) guidance does not replace official NWS Aviation products such as the TAF. Several sites available on LAMP do not have TAFs. Use at your own risk. Always consult official aviation weather products for flight planning and operational decisions.

Data Values

  • 999 = Not part of guidance (NOAA does not provide)
  • % suffix = Probability/Percentage value
  • - = Missing or unavailable data

Temperature Colors

  • Temperature = Highlighted for extreme values:
    • Blue = ≤32°F (freezing)
    • Red = ≥85°F (hot)
  • Apparent Temperature (Feels-Like) = Shown in station local time using wind chill danger coloring:
    • Blue = 0°F or colder (cold)
    • Yellow = -20 to -34°F (30-min frostbite risk)
    • Orange = -35 to -49°F (10-min frostbite risk)
    • Red = -50°F or colder (5-min extreme danger)

Flight Categories

  • VFR = Visual Flight Rules (best conditions)
  • MVFR = Marginal VFR
  • IFR = Instrument Flight Rules
  • LIFR = Low IFR (worst conditions)

Highlighted Values

  • Yellow highlight = Significant value (≥50% probability, ≥20kt wind, ≥90% humidity, ≤3°F spread, ≥3/10 convection)
  • Gray cell = Current METAR observation
